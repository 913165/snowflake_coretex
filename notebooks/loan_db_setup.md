# LOAN_DB Setup Guide

## Database & Schema
```sql
CREATE OR REPLACE DATABASE LOAN_DB;
CREATE SCHEMA IF NOT EXISTS LOAN_DB.LENDING;
```

## Tables

### CUSTOMERS
```sql
CREATE OR REPLACE TABLE LOAN_DB.LENDING.CUSTOMERS (
    CUSTOMER_ID INT,
    FIRST_NAME VARCHAR(50),
    LAST_NAME VARCHAR(50),
    EMAIL VARCHAR(100),
    PHONE VARCHAR(20),
    DATE_OF_BIRTH DATE,
    ANNUAL_INCOME NUMBER(12,2),
    EMPLOYMENT_STATUS VARCHAR(20),
    CREDIT_SCORE INT,
    STATE VARCHAR(2),
    CREATED_AT TIMESTAMP_NTZ DEFAULT CURRENT_TIMESTAMP()
);
```

### LOANS
```sql
CREATE OR REPLACE TABLE LOAN_DB.LENDING.LOANS (
    LOAN_ID INT,
    CUSTOMER_ID INT,
    LOAN_TYPE VARCHAR(30),
    LOAN_AMOUNT NUMBER(12,2),
    INTEREST_RATE NUMBER(5,3),
    TERM_MONTHS INT,
    ORIGINATION_DATE DATE,
    MATURITY_DATE DATE,
    LOAN_STATUS VARCHAR(20),
    PURPOSE VARCHAR(50),
    MONTHLY_PAYMENT NUMBER(10,2)
);
```

### LOAN_APPLICATIONS
```sql
CREATE OR REPLACE TABLE LOAN_DB.LENDING.LOAN_APPLICATIONS (
    APPLICATION_ID INT,
    CUSTOMER_ID INT,
    APPLICATION_DATE DATE,
    REQUESTED_AMOUNT NUMBER(12,2),
    APPROVED_AMOUNT NUMBER(12,2),
    DECISION VARCHAR(20),
    DECISION_DATE DATE,
    DEBT_TO_INCOME_RATIO NUMBER(5,3),
    LOAN_TO_VALUE_RATIO NUMBER(5,3)
);
```

### PAYMENTS
```sql
CREATE OR REPLACE TABLE LOAN_DB.LENDING.PAYMENTS (
    PAYMENT_ID INT,
    LOAN_ID INT,
    PAYMENT_DATE DATE,
    AMOUNT_DUE NUMBER(10,2),
    AMOUNT_PAID NUMBER(10,2),
    PAYMENT_STATUS VARCHAR(20),
    DAYS_LATE INT DEFAULT 0
);
```

- **CUSTOMERS** (1) --< (many) **LOANS** via `CUSTOMER_ID`
